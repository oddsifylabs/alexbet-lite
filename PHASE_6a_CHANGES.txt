JAVASCRIPT FUNCTION CHANGES

NEW FUNCTION: getStatCardType(metricKey)
Purpose: Maps metric keys to semantic card types
Signature: function getStatCardType(metricKey) → string
Returns: 'success' | 'error' | 'warning' | 'info'

Metric Mappings:
  - winRate → 'success'
  - wins, totalWins → 'success'
  - totalPnL (positive) → 'success', (negative) → 'error'
  - roi (positive) → 'success', (negative) → 'error'
  - avgEdge (positive) → 'success', (negative) → 'error'
  - totalLosses, losses → 'error'
  - totalWagered → 'warning'
  - clv, confidence → 'info'
  - status (contains ✅/ON TRACK) → 'success'
  - status (contains ⚠️/BELOW) → 'error'

COLOR-CODED METRIC SYSTEM

  Used for: Win Rate, Profit (when positive), Edge (when positive), Status (on track)
  Colors: #00d68f (bright), rgba(0, 214, 143, 0.08) (background)

🔴 ERROR (Red) — Negative/Bad Metrics
  Used for: Losses, Profit (when negative), Edge (when negative), Status (below target)
  Colors: #ff6464 (bright), rgba(255, 100, 100, 0.08) (background)

🟠 WARNING (Orange) — Neutral/Money Metrics

  Colors: #ffa726 (bright), rgba(255, 167, 38, 0.08) (background)

  Colors: #4d9fff (bright), rgba(77, 159, 255, 0.08) (background)
