# AlexBET Lite

Standalone bet tracker & analytics app. Test and validate the gem algorithm before launching to public.

## Features

- **Bet Tracker** — Log picks, odds, edge, stake, sport
- **Real-time Analytics** — Win rate, P&L, ROI, avg edge
- **Auto-Settle** — Mark bets as WON/LOST/PUSH
- **Validation** — Track progress toward 56-65% win rate target
- **Export** — Download bet history as JSON

## Target

✅ **56-65% win rate** = Ready to launch

## Usage

1. Open `index.html` in browser
2. Add bets as you place them from the bot
3. Mark results when games settle
4. Watch win rate in real-time
5. Export data for analysis

## Data Storage

All bets stored in browser's LocalStorage — persists across sessions.

## Deploy

```bash
git push origin main
```

Then deploy to Netlify, GitHub Pages, or Vercel as static site.
